O que é HTML 5?

HTML5 (Hypertext Markup Language, versão 5) é uma linguagem para estruturação e apresentação de conteúdo para a World Wide Web e é uma tecnologia chave da Internet. É a quinta versão da linguagem HTML.

Qual meus conhecimentos em HTML 5?

Meu conhecimento em HTML 5 é avançado. Domino toda a estrutura do HTML, os elementos e atributos.
Ao inicio de um projeto, sendo ele pequeno ou não, faço um planejamento sobre a estrutura da interface, para ter em mente as tags que utilizarei, dando total importância na semâtica do código, obtendo assim código limpo e rápido entendimento.
